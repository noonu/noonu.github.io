配置参数

名称 类型 默认值 描述
width Number 800 整体宽度
height Number 400 整体高度
barWidth Number 30 控制条宽度
showInBar Boolean false 控制条是否也显示内容区左边部分内容
initialActiveItem Number 0 初始时显示项(编号是从零计数的)
speed Number 500 滑动速度（滑动的时间ms）
easing String 'swing' 滑动的效果
onTriggerAnimate Function Null 滑动开始时调用的函数
onAnimateComplete Function Null 滑动结束时调用的函数
autoPlay Boolean false 是否自动播放
pauseOnHover Boolean true 鼠标hover的时候是否暂停
playInterval Number 800 自动播放的时间间隔

扩展方法

名称 描述
start() 启动播放
stop() 停止播放
start() 启动播放
next() 播放到下一项
prev() 播放到前一项
activeItem(number) 播放到指定项(编号是从零计数的)

扩展属性

名称 描述
itemToShow 代表即将要显示的项(一个li,纯粹的DOM对象，不是jQuery实例对象)
itemToHide 代表即将被隐藏的项(一个li,纯粹的DOM对象，不是jQuery实例对象)

详细使用说明

调用方法
$("#accordion-wrap").tinyAccordion({...});
将手风琴的容器div获取为一个jQuery对象，然后调用tinyAccordion()方法即可，通常需要向tinyAccordion方法传递一个对象作为参数。（请参考参数说明）
html结构要求 
<div>
  <ul>
    <li>
      <h2></h2>
      <div></div>
    </li>
    <li>
      <h2></h2>
      <div></div>
    </li>
    ...... ......
  </ul>
</div>
解释：最外层div是手风琴的整体，对其id和class没有要求，方 便获取就好了。里面是一个ul，li就是各个子项，你可以按自己 需求放置制任意项数，子项里面的h2为左边的控制条，后面的div为内容块，可以自己安排放置你想放的任意内容
CSS要求
CSS没有要求，当然你也可以自己定义样式，特别是定义内容div
调用参数说明
可以传入一个对象作为参数，可以不设参数对象的某个属性， 不设的话将使用默认值!
接口方法调用方式
为了方便插件使用者通过其他代码来控制手风琴，如通过点击某个按钮停止 /启动循环播放、播放下一项/上一项，可以在手风琴对象上调用如下方法，调用方式类似于：
var $accordion = $("#accordion-wrap").tinyAccordion({...});
$("#button").click(function() { $accordion.next(); });
其他说明
1、滑动的触发方式，默认是click，可以设置为mouseover。
2、滑动效果：由于jquery默认可用的只有linear和swing，如果你想实现 其他效果，请下载其他easing，如jqueryUI里面有提供。
3、两个回调函数的执行环境(即里面的this指向)被设为包裹手风琴整体的div，如此可以方便的控制手风琴。
4、如果希望取得itemToShow(即将显示出来的项)或itemToHide(原来显示,但即将被遮挡的项)，可以在手风琴上以属性的方式来访问。比如在上面说 的两个回调中，通过this.itemToSow、this.itemToHide就可以访问了。 (注意：它们是纯粹的DOM对象，不是jQuery实例对象)
